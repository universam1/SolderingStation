# About:
library shield_arduino 

(for arduino board version make from 2009 and after)


#Contains: 
the KiCAD schematic for some "ARDUINO" board:

ARDUINO_as_UNO		schématic for build a shield or clone an "arduino as UNO " based on revision 3
ARDUINO_FIO		schématic for build a shield or clone an "arduino FIO" based on version 22
ARDUINO_MEGA		schématic for build a shield or clone an "arduino MEGA 2560 " based on revision 3 
ARDUINO_MICRO		schématic for build a shield or clone an "arduino MICRO " based on revision 3
ARDUINO_MINI_PRO	schématic for build a shield or clone an "arduino PRO MINI " based on version 10
ARDUINO_MINI		schématic for build a shield or clone an "arduino MINI " based on version 5
ARDUINO_NANO		schématic for build a shield or clone an "arduino NANO" based on version 2

/!\Attention/!\ 

GND/1, GND/2 ou TX/1, TX/2 ou 5V/1, 5V/2 ect... as the same in your board but it was separate in these modules.
Just for more free during your construction.

Schematics not contains the ISP connector.

Pin are placing like your official boards

Just verifi if pins are present, depending your board version and adjust in consequence.

Be careful the library may contain errors.



(partie en français)

/!\Attention/!\ 

Les pins marquées GND/1, GND/2 ou TX/1, TX/2 ou 5V/1, 5V/2 ect... sont connectées 
entre elles sur les cartes officielles. 
Sur les schémats je l'ai est dissocié pour vous laissez plus de 
liberté dans la fabriction de vos shields ou vos clones...

Les schémas ne contiennent pas le connecteur ISP.

L'arrangement des connections sur le schémat et suivant le placement des pins sur
les cartes officielles 

Vérifier la version de votre carte et la présence des pins. 
Ajuster en conséquence.

La librairie peut contenir des défauts alors vérifier deux fois.
