All view are made in "inch" with any export offset.

The directory "demo_library_wings" contains all modules 
3D for demonstration, you can use it for select your 3D view.

File "extra.wings" contain all pins i used and little more in format ".wings".
You can use for create your personnal view.
 
For open you need the software -> "wings 3D" 
You can download at : http://www.wings3d.com/

Be careful the library may contain errors



(partie en français)

Toutes les vues sont construites en "pouce" sans offset à l'exportation.

Le répertoire "demo_library_wings" contient tous l'ensemble des modules 3D.
Vous pouvez vous en servir pour visualiser et choisir votre vue 3D.

Le fichier "extra.wings" contient les éléments 3D au format ".wings" que j'ai utilisé pour
faire les modules. Vous pouvez ainsi adapter vos vues selon vos besoins.

Pour utiliser les fichier ".wings" il vous faut le logiciel "wings 3D"
que vous pouvez trouver ici: http://www.wings3d.com/

Attention les vues peuvent contenir des défauts alors vérifier deux fois.
